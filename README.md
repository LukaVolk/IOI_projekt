# IOI_projekt

## PM10 Air Quality Dashboard

This project is a web application for visualizing PM10 air quality data from monitoring stations across Slovenia.

### Usage

1. Prepare your data files:
	- **measurements_data.csv**: Required. Upload this file using the web app interface to visualize air quality data. You can find this file in the `data` directory of the project.

2. Access the web application here:
	- [https://air-slovenia-monitor.vercel.app/](https://air-slovenia-monitor.vercel.app/)

3. Upload your data files as prompted in the sidebar to begin exploring the data.
